TanMenu Portable / 绿色版
=========================

中文
----
1. 将整个文件夹解压到一个可写位置，例如 D:\Apps\TanMenu。
2. 运行 TanMenu.Wpf.exe。
3. 配置、主题、缓存、日志和 WebView2 用户数据都保存在同目录的 Data 文件夹。
4. portable.flag 是绿色模式标记，请勿删除；删除后程序会按普通版规则使用“文档\TanMenu”。
5. 绿色版不会写入开机启动注册表，因此设置中的“开机自启”不可用。
6. 卸载时退出 TanMenu 并删除整个文件夹即可。

TanMenu 需要 Microsoft Edge WebView2 Runtime。Windows 11 通常已经预装；如果启动时提示
缺少运行时，请从 https://developer.microsoft.com/microsoft-edge/webview2/ 安装 Evergreen Runtime。

English
-------
1. Extract the complete folder to a writable location, for example D:\Apps\TanMenu.
2. Run TanMenu.Wpf.exe.
3. Config, themes, caches, logs, and WebView2 user data stay in the adjacent Data folder.
4. portable.flag enables portable mode. Do not remove it; without it TanMenu uses the normal
   Documents\TanMenu data-location policy.
5. The portable edition does not write a Windows startup registry entry, so autostart is unavailable.
6. To uninstall, exit TanMenu and delete the extracted folder.

Microsoft Edge WebView2 Runtime is required. It is normally preinstalled on Windows 11. If TanMenu
reports that it is missing, install the Evergreen Runtime from the URL above.
