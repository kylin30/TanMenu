## TanMenu 0.9.2

- 绿色版新增自动更新：启动后从 GitHub Releases 静默检查并下载，在“设置 > 更新”中确认重启即可安装。
- 绿色版数据改存于更新安全的根目录 `Data` 文件夹，更新程序不会覆盖配置、主题、日志或 WebView2 数据。
- Microsoft Store 版本仍由 Microsoft Store 管理更新，不使用 GitHub 更新通道。

> 从 0.9.1 升级时需要最后一次手动下载本版本的绿色 ZIP。请先退出 TanMenu，再将 ZIP 解压到原绿色版目录，并保留原有 `Data` 文件夹；从 0.9.2 开始，后续版本可在应用内更新。

---

- The portable edition now checks and downloads updates from GitHub Releases. Confirm the restart under Settings > Updates to install one.
- Portable data now lives in the update-safe root `Data` folder and is preserved across updates.
- The Microsoft Store edition remains managed by Microsoft Store.

Users upgrading from 0.9.1 need to download this portable ZIP once. Exit TanMenu, extract it into the
existing portable directory, and keep the existing `Data` folder. In-app updates work from 0.9.2 onward.
